📖 Definición de Números Racionales

Un número racional es aquel que puede expresarse como el cociente de dos números enteros, con denominador distinto de cero.

$$\mathbb{Q} = \left\{ \frac{a}{b} \mid a, b \in \mathbb{Z}, b \neq 0 \right\}$$
🎯 Elementos:
• $a$ → numerador (puede ser cualquier entero)
• $b$ → denominador (entero distinto de cero)
💡 Ejemplos de números racionales:
$\frac{1}{2}, \frac{3}{4}, -\frac{5}{3}, 7 = \frac{7}{1}, 0 = \frac{0}{1}, 0.\overline{3} = \frac{1}{3}$
2 / 16

🔢 El conjunto $\mathbb{Q}$

Los números racionales incluyen a otros conjuntos numéricos:

📌 Naturales ($\mathbb{N}$)

$1, 2, 3, 4, ...$
Todos son racionales: $1 = \frac{1}{1}$

📌 Enteros ($\mathbb{Z}$)

$..., -3, -2, -1, 0, 1, 2, 3, ...$
$ -5 = \frac{-5}{1}$

📌 Fracciones

$\frac{1}{2}, \frac{3}{4}, -\frac{2}{5}, \frac{7}{3}$

📌 Decimales exactos

$0.5 = \frac{1}{2}$, $0.75 = \frac{3}{4}$, $2.5 = \frac{5}{2}$

📌 Decimales periódicos

$0.\overline{3} = \frac{1}{3}$, $0.\overline{6} = \frac{2}{3}$, $1.\overline{6} = \frac{5}{3}$

💡 Propiedad: Entre dos números racionales siempre existe otro número racional. ¡Son infinitos y densos!

🔄 Transformar fracción a decimal

Para convertir una fracción a decimal, se divide el numerador por el denominador.

📌 Ejemplo 1: $\frac{3}{4} = 3 \div 4 = 0.75$ (decimal finito)
📌 Ejemplo 2: $\frac{1}{3} = 1 \div 3 = 0.3333... = 0.\overline{3}$ (decimal periódico)
📌 Ejemplo 3: $\frac{7}{8} = 7 \div 8 = 0.875$ (decimal finito)
📌 Ejemplo 4: $\frac{5}{6} = 5 \div 6 = 0.8333... = 0.8\overline{3}$ (decimal semiperiódico)

🔄 Transformar decimal a fracción

📌 Decimal finito

$0.75 = \frac{75}{100} = \frac{3}{4}$
Se escribe el número sin coma como numerador y una potencia de 10 como denominador.

📌 Decimal periódico puro

$0.\overline{3} = \frac{3}{9} = \frac{1}{3}$
Numerador: el período. Denominador: tantos 9 como cifras tenga el período.

📌 Decimal semiperiódico

$0.8\overline{3} = \frac{83-8}{90} = \frac{75}{90} = \frac{5}{6}$
Numerador: parte completa menos parte no periódica. Denominador: tantos 9 como cifras del período seguidos de tantos 0 como cifras de la parte no periódica.

📊 Decimales finitos e infinitos

✅ Decimal finito (exacto)

La división termina. Ej: $\frac{1}{2}=0.5$, $\frac{3}{4}=0.75$, $\frac{1}{8}=0.125$

Una fracción irreducible genera un decimal finito si su denominador solo tiene factores primos 2 y 5.
🔄 Decimal infinito periódico

Un dígito o grupo se repite indefinidamente. Ej: $\frac{1}{3}=0.\overline{3}$, $\frac{2}{7}=0.\overline{285714}$

🔁 Decimal infinito semiperiódico

Hay una parte no periódica seguida de un período. Ej: $\frac{5}{6}=0.8\overline{3}$

⚠️ Nota: Los números irracionales (como $\pi$ o $\sqrt{2}$) tienen decimales infinitos NO periódicos. Estos NO pertenecen a $\mathbb{Q}$.

➕ Adición y sustracción (igual denominador)

$$\frac{a}{c} + \frac{b}{c} = \frac{a+b}{c}$$ $$\frac{a}{c} - \frac{b}{c} = \frac{a-b}{c}$$

Cuando las fracciones tienen el mismo denominador, se suman o restan los numeradores y se conserva el denominador.

📌 Ejemplo 1: $\frac{3}{7} + \frac{2}{7} = \frac{3+2}{7} = \frac{5}{7}$
📌 Ejemplo 2: $\frac{5}{9} - \frac{2}{9} = \frac{5-2}{9} = \frac{3}{9} = \frac{1}{3}$
⚠️ Recuerda: Simplificar el resultado siempre que sea posible.

➕ Adición y sustracción (distinto denominador)

Para sumar o restar fracciones con diferente denominador:

  1. Se calcula el mínimo común múltiplo (MCM) de los denominadores.
  2. Se amplifican las fracciones para que tengan el mismo denominador (el MCM).
  3. Se suman o restan los numeradores y se conserva el denominador.
$$\frac{a}{b} + \frac{c}{d} = \frac{a \cdot d + c \cdot b}{b \cdot d}$$
📌 Ejemplo: $\frac{2}{3} + \frac{1}{4} = \frac{2 \cdot 4 + 1 \cdot 3}{3 \cdot 4} = \frac{8+3}{12} = \frac{11}{12}$
📌 Ejemplo 2: $\frac{5}{6} - \frac{1}{4} = \frac{5 \cdot 4 - 1 \cdot 6}{6 \cdot 4} = \frac{20-6}{24} = \frac{14}{24} = \frac{7}{12}$

✖️ Multiplicación de números racionales

$$\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d} \quad (b \neq 0, d \neq 0)$$

Para multiplicar fracciones, se multiplican los numeradores y multiplican los denominadores.

📌 Ejemplo 1: $\frac{2}{3} \times \frac{4}{5} = \frac{2 \times 4}{3 \times 5} = \frac{8}{15}$
📌 Ejemplo 2: $\frac{3}{4} \times \frac{5}{6} = \frac{3 \times 5}{4 \times 6} = \frac{15}{24} = \frac{5}{8}$ (simplificando antes: $\frac{3}{4} \times \frac{5}{6} = \frac{1}{4} \times \frac{5}{2} = \frac{5}{8}$)
💡 Simplificación cruzada: Puedes simplificar numeradores con denominadores antes de multiplicar.

➗ División de números racionales

$$\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} = \frac{a \times d}{b \times c} \quad (b \neq 0, c \neq 0, d \neq 0)$$

Para dividir fracciones, se multiplica la primera por el recíproco (inverso multiplicativo) de la segunda.

📌 Ejemplo 1: $\frac{2}{3} \div \frac{4}{5} = \frac{2}{3} \times \frac{5}{4} = \frac{2 \times 5}{3 \times 4} = \frac{10}{12} = \frac{5}{6}$
📌 Ejemplo 2: $\frac{3}{4} \div 2 = \frac{3}{4} \div \frac{2}{1} = \frac{3}{4} \times \frac{1}{2} = \frac{3}{8}$
📌 Ejemplo 3: $5 \div \frac{2}{3} = \frac{5}{1} \times \frac{3}{2} = \frac{15}{2} = 7.5$
⚠️ Importante: No se puede dividir por cero. El divisor (segunda fracción) debe ser distinto de cero.

🔢 Operaciones combinadas con racionales

Al igual que con los enteros, se respeta el orden de las operaciones (PEMDAS):

  1. Paréntesis
  2. Exponentes / Potencias
  3. Multiplicación y División (de izquierda a derecha)
  4. Adición y Sustracción (de izquierda a derecha)
📌 Ejemplo: $\frac{2}{3} + \frac{1}{2} \times \frac{4}{5} = \frac{2}{3} + \frac{4}{10} = \frac{2}{3} + \frac{2}{5} = \frac{10}{15} + \frac{6}{15} = \frac{16}{15}$
📌 Ejemplo con paréntesis: $\left(\frac{1}{2} + \frac{1}{3}\right) \times \frac{3}{5} = \left(\frac{3}{6} + \frac{2}{6}\right) \times \frac{3}{5} = \frac{5}{6} \times \frac{3}{5} = \frac{15}{30} = \frac{1}{2}$

📐 Propiedades de la operatoria en $\mathbb{Q}$

🔹 Clausura

La suma, resta, multiplicación y división (por ≠0) de racionales da otro racional.

🔹 Conmutatividad

$a+b = b+a$, $a \times b = b \times a$

🔹 Asociatividad

$(a+b)+c = a+(b+c)$
$(a \times b) \times c = a \times (b \times c)$

🔹 Distributividad

$a \times (b+c) = a \times b + a \times c$

🔹 Elemento neutro

$a+0 = a$, $a \times 1 = a$

🔹 Inverso

$a + (-a) = 0$, $a \times \frac{1}{a} = 1$ (para $a \neq 0$)

✏️ Ejemplos resueltos paso a paso

1️⃣ Suma: $\frac{3}{8} + \frac{5}{12} = \frac{3 \cdot 3 + 5 \cdot 2}{24} = \frac{9+10}{24} = \frac{19}{24}$
MCM(8,12) = 24
2️⃣ Multiplicación: $\frac{7}{9} \times \frac{3}{14} = \frac{7 \times 3}{9 \times 14} = \frac{21}{126} = \frac{1}{6}$
Simplificación: 7 con 14, 3 con 9
3️⃣ División: $\frac{5}{6} \div \frac{10}{3} = \frac{5}{6} \times \frac{3}{10} = \frac{15}{60} = \frac{1}{4}$
4️⃣ Combinada: $\frac{2}{5} \times \left(\frac{1}{3} + \frac{1}{2}\right) = \frac{2}{5} \times \frac{5}{6} = \frac{10}{30} = \frac{1}{3}$
Primero se suma dentro del paréntesis: $\frac{1}{3} + \frac{1}{2} = \frac{5}{6}$

🌍 Aplicaciones de los números racionales

💰 Finanzas

Porcentajes, intereses, reparto de utilidades, IVA ($19\% = \frac{19}{100}$)

🍳 Cocina

Recetas: $\frac{1}{2}$ taza, $\frac{3}{4}$ cucharadita

📐 Medidas

Longitudes: $\frac{1}{2}$ metro, $\frac{3}{8}$ pulgada

⚡ Física

Velocidades medias, densidades, fracciones de tiempo

🎯 Probabilidad

Probabilidad de un evento: $\frac{1}{6}$ al lanzar un dado

📊 Estadística

Proporciones, frecuencias relativas, porcentajes

🏆 Resumen final - Números Racionales

✅ Definición: $\mathbb{Q} = \{\frac{a}{b} \mid a,b \in \mathbb{Z}, b \neq 0\}$
✅ Operaciones clave:
Suma/resta: Mismo denominador → sumar numeradores
Multiplicación: $\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}$
División: $\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}$
✅ Decimales:
Finito: denominador solo factores 2 y 5
Periódico: se repite un patrón
